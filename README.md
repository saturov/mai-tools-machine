# my-tools-sandbox

Песочница для экспериментов с утилитами, схемами и шаблонами, которые используются в моих агентах/воркфлоу.

## Содержимое

- `docs/` — архитектура и заметки.
- `schemas/` — JSON-схемы для контрактов/манифестов.
- `templates/` — шаблоны для генерации/скелетов.

См. `docs/skills-ecosystem-architecture.md` для описания целевой архитектуры экосистемы утилит и скиллов.

## Статус

Репозиторий в активной разработке. Папки `tg-scraper/` и `yb-downloader/` могут существовать локально, но пока не считаются частью публичной истории (не коммитятся).
